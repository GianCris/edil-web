# EDIL Web

Página web desarrollada para un negocio real de ropa personalizada (EDIL), enfocada en mostrar productos, servicios y facilitar la comunicación con clientes mediante WhatsApp.

## Descripción

Este proyecto consiste en una landing page donde se presentan los principales servicios del negocio:

- Bordado personalizado
- DTF (estampado)
- Prendas personalizadas
- Productos propios de la marca EDIL
- Diseños en tendencia

La web permite a los clientes visualizar productos y realizar consultas o pedidos directamente por WhatsApp.

## Tecnologías utilizadas

- HTML
- CSS

## Objetivo del proyecto

Este proyecto fue desarrollado como parte de mi portafolio, aplicando programación para resolver necesidades reales de un negocio, como la presentación de productos y la captación de clientes.
